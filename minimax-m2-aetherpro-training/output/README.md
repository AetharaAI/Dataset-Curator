## Format Specification

Each JSONL line contains a JSON object with:

```json
{
  "text": "<|user|>\\n{USER_MESSAGE}\\n<|end|>\\n<|assistant|>\\n<think>\\n{THINKING_TRACE}\\n</think>\\n{RESPONSE}\\n<|end|>",
  "source": "gemini_validation",
  "category": "first_principles",
  "quality_score": 9
}
```

### Text Field Structure
- Conversation format with `<|user|>` and `<|assistant|>` tags
- Thinking traces wrapped in `<think>` tags
- Turns end with `<|end|>` tags
- Multi-turn conversations supported

### Quality Score Criteria (1-10)
- 10: Equation + real constraint + failure mode + multi-turn
- 8-9: 3 of above + substantial thinking (200+ chars)
- 6-7: 2 of above + technical accuracy
- 4-5: 1 of above, technically correct
- 1-3: Missing principles, shallow, or repetitive

## Usage

To use in training:
```python
import json

examples = []
with open('training_dataset.jsonl') as f:
    for line in f:
        example = json.loads(line)
        examples.append(example)

# Filter by category
technical = [ex for ex in examples if ex['category'] in ['first_principles', 'electrical']]

# Filter by quality
high_quality = [ex for ex in examples if ex['quality_score'] >= 8]
```
